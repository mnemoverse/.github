# Mnemoverse Organization Configuration

This repository contains organization-wide configuration files and community health documents for the Mnemoverse organization.

## Contents

- `profile/README.md` - Organization profile displayed on https://github.com/mnemoverse
- `CODE_OF_CONDUCT.md` - Community guidelines
- `CONTRIBUTING.md` - Contribution guidelines  
- `SECURITY.md` - Security policy
- `.github/ISSUE_TEMPLATE/` - Issue templates for all repositories
- `scripts/` - Automation scripts for repository management

## Usage

These files are automatically applied to all public repositories in the organization unless overridden by repository-specific files.
